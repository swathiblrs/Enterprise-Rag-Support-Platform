# Duo MFA Knowledge Base

## Overview
This article explains how to troubleshoot Duo Multi-Factor Authentication issues.

## Common Symptoms
Users may report that they are not receiving Duo push notifications, cannot approve MFA, changed phones, or lost access to their authentication device.

## Troubleshooting Steps
1. Verify that the user has network connectivity on their mobile device.
2. Ask the user to open the Duo Mobile app manually.
3. Ask the user to check whether push notifications are enabled.
4. If the user changed phones, re-register the new device.
5. If the user lost the device, verify identity and issue a temporary bypass code according to policy.

## Escalation
If the user cannot complete MFA after basic troubleshooting, route the ticket to Identity and Access Management.
