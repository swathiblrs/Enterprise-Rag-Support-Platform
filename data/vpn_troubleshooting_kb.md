# VPN Troubleshooting Knowledge Base

## Overview
This article explains how to troubleshoot VPN connectivity issues for enterprise users.

## Common Symptoms
Users may report that the VPN disconnects frequently, fails after a password reset, shows an authentication error, or cannot connect to the company network.

## Troubleshooting Steps
1. Confirm the user can log in to the Single Sign-On portal.
2. Ask the user to restart the VPN client.
3. Verify that Duo MFA approval is working.
4. Clear saved VPN credentials from the VPN client.
5. Confirm the user is using the latest VPN client version.
6. Ask the user to reconnect using the new password if the password was recently changed.

## Escalation
If the issue continues after these steps, route the ticket to Network Support.
